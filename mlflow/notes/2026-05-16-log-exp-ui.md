# Logging My First Experiment and Finding it in the MLflow UI

I logged my first experiment today. Here's how I found it.

## Log an Experiment

Used the Python API:
```python
import mlflow

mlflow.set_experiment("my-first-exp")

with mlflow.start_run() as run:
    mlflow.log_param("learning_rate", 0.01)
    mlflow.log_metric("loss", 0.523)
    print(f"Run ID: {run.info.run_id}")
```

## Find it in the UI

Started the UI:
```bash
mlflow ui
```

Then open http://127.0.0.1:5000

The experiment "my-first-exp" appears in the left sidebar. Click it to see runs with params and metrics.

## From CLI

```bash
mlflow runs list -e my-first-exp
```

Output shows run IDs, status, and timestamps.
